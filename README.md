## 所需拓展
* **[RIPEXT](https://github.com/ErikMinekus/sm-ripext/releases/tag/1.3.1)**
## 功能
- **🎇 浮游炮本体功能**
- **🏆 高级功能**
  - Convar设置（浮游炮API、弹道材质、Prefix）
  - 玩家指令（浮游炮面板、更改浮游炮颜色、强制刷新浮游炮）
  - **卡密兑换**
  - 自定义颜色（弹道及炮体）
  - （服务器功能）插件云端验证
